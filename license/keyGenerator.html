License Key Generator

Developer tool — do not distribute. Generate Ed25519 key pairs and sign license keys.

Step 1 — Key Pair

First time? Click "Generate New Key Pair" below. Copy the Public Key into licenseValidator.js → PUBLIC_KEY_B64, then save the Private Key JWK somewhere secure (password manager or encrypted file). Next time, paste your saved private key into the field below — do not generate a new pair, or all old license keys will stop working.
